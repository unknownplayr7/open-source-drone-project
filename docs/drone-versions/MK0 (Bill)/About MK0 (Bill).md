# MK0 System Definition

MK0 represents a conceptual 400mm quadcopter platform used to define the structural and architectural layout of the drone system.

It serves as a non-operational reference model for:

- component placement
- structural layout
- weight distribution estimation
- modular design validation

MK0 is not intended to be physically assembled or flown.
## MK0 Completion Criteria

MK0 is considered complete when:

- A full system layout has been defined (frame, power, control, payload zones)
- Estimated weight range is established
- Center of mass placement is conceptually validated
- Component categories are selected (not final parts)
- CAD mockup of frame exists (non-final geometry)
- Design constraints are documented and finalized
- No hardware integration has been performed

## MK0 does not include physical assembly or electronics testing.
